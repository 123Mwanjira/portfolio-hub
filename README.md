# PortfolioHub - React SPA

## Project Description
PortfolioHub is a single-page application built with React and Vite. It allows users to view, search, and add portfolio projects dynamically, and is designed to showcase creative work in a modern interface.

---

## Features

- Display a list of projects
- Add new projects dynamically
- Search and filter projects in real time
- Responsive design for all screen sizes
- Clean, modern UI

---

## Technologies Used

- React (Vite)
- JavaScript (ES6+)
- CSS3
- Git & GitHub

## Setup Instructions

1. Clone repository
   ```bash
git clone https://github.com/your-username/portfolio-hub.git
cd portfolio-hub
```
2. Install dependencies
   ```bash
npm install
```
3. Run the app
   ```bash
npm run dev
```

## Testing

Run tests locally:
```bash
npm test
```

If your Windows path contains spaces, `npm test` now detects that and automatically falls back to running the temporary copy workaround.
You can also run the fallback directly:
```bash
npm run test:copy
```
