 PortfolioHub - React SPA

 Project Description
PortfolioHub is a Single Page Application (SPA) built with React that allows users to view, search, and add portfolio projects dynamically. It is designed for a creative agency to showcase their work in a modern and responsive interface.

---

Features

- Display list of projects
- Add new projects dynamically
- Search/filter projects in real time
- Responsive design for all screen sizes
- Clean and modern UI

---

 Technologies Used

- React (Vite)
- JavaScript (ES6+)
- CSS3
- Git & GitHub
